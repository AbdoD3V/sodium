# Replace the directory (linux) of this readme file with the official linux kernel folder.

You can clone the Linux kernel using git, make sure it's the official one.
in the root folder, run `git clone --depth 1 https://github.com/torvalds/linux.git`, wait for the kernel to finish cloning, it'll clone into this directory

and don't forget to compile your linux kernel however you want, you can use `make defconfig` for the default configuration then run `make`
